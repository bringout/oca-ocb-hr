# Skills Certification


Certification and Skills for HR
===============================

This module adds certification to resume for employees.
        

## Installation

```bash
pip install odoo-bringout-oca-ocb-hr_skills_survey
```

## Dependencies

This addon depends on:
- hr_skills
- survey
