# Timesheets/attendances reporting


    Module linking the attendance module to the timesheet app.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-hr_timesheet_attendance
```

## Dependencies

- hr_timesheet
- hr_attendance
