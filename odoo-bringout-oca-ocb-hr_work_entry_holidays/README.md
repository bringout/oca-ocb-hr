# Time Off in Payslips


Manage Time Off in Payslips
============================

This application allows you to integrate time off in payslips.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-hr_work_entry_holidays
```

## Dependencies

- hr_holidays
- hr_work_entry
